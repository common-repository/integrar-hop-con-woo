== Descripcion ==
Con este plugin vas a poder obtener costos de envio para WooCommerce utilizando la API publica de HOP Envios.

 
== Instalacion ==

1. Install the WooCommerce HOP Envios - Plugin.
2. Activate the plugin.
3. Go to the WooCommerce Settings Page. 
4. Access Shipping Tab.
5. Select "Shipping Zones" and Add HOP Envios.
6. Check the Enable/Disable Checkbox.
7. Enter the settings that you would like you to use.
8. Enter the dimensions of the products, (HOP Envios uses this to determine suitability of products).
